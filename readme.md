# MusicDownloader

音乐下载工具，目前支持网易云音乐、QQ音乐。仅供学习研究使用。

## 功能

- 搜索关键词，返回支持站点的歌曲列表。
- 根据 ID 获取网易云音乐歌单。
- 过滤不能播放的歌曲（灰色）
- 下载、播放列表中的歌曲，如果有歌词，默认一起下载。

## 依赖

- PyQt5
- requests
- bs4
- Crypto（网易云音乐）
- execjs（歌单）

## 注意事项

- 播放 QQ 音乐的歌曲需要安装 [LAVFilters](https://github.com/Nevcairiel/LAVFilters/releases)
- 音质为不登录时的普通音质
- 本代码不直接提供下载
